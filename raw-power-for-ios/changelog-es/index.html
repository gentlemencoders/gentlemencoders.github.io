RAW Power for iOS Release Notes

Version 3.4.12:

  • (iOS 14+) Añadir soporte para Sony A7 R5. Ver lista completa
  • Nuevo: Añadir palabras clave y descripción al exportar.
  • Añadir solución de problemas e información de apoyo cuando se utiliza el almacenamiento externo.
  • Corrección: dos fallos.
  • Corrección: las imágenes editadas externamente se exportaban como imágenes sin ajustar.
