RAW Power pour Mac FAQ

Que font les commandes Linéaire / Gamma et Luminance / RVB égaux dans Courbes ?

Le nom affiché sur les boutons indique l’état actuel du contrôle. Si vous voyez «Linéaire», Courbes fonctionne en mode Linéaire, mais si vous voyez Gamma, Courbes traite en mode corrigé gamma. Cela est également vrai pour le bouton Luminance / RVB égaux.

RAW Power fonctionne avec des données de pixels linéaires à virgule flottante. Si vous choisissez Linéaire pour Courbes, Courbes manipulera les données de pixels linéaires telles quelles. Si vous choisissez Gamma, Curves corrigera les pixels de manière gamma, appliquera la courbe, puis annulera la correction de gamma (pour que les données de pixel soient reconverties en valeurs linéaires).

RVB égaux signifie que la courbe combinée appliquera les mêmes modifications aux composants rouge, vert et bleu de chaque pixel. Cela affectera n’importe quelle teinte de l’image. Cependant, si vous choisissez Luminance, la courbe combinée modifiera la valeur de luminance de chaque pixel et n’affectera pas la teinte.

Quelle est la différence entre Netteté RAW et le réglage Netteté?

Netteté RAW (présent dans l’ajustement Traitement RAW) est un renforcement subtil qui fonctionne dans le cadre du décodage RAW lui-même. L’ajustement Netteté est un dispositif de netteté orienté luminance qui s’exécute après le décodage, mais qui fonctionne toujours sur les données linéaires trouvées dans les fichiers RAW.

Quelle est la meilleure façon d’utiliser l’échantillonneur de balance des blancs?

Il est préférable d’identifier dans votre image une zone de taille raisonnable qui devrait être un gris neutre. Ne choisissez pas une zone blanche et brillante.

Que fait la Récupération?

La récupération fonctionne comme une «exposition sélective». Elle affecte les zones les plus lumineuses, mais sans toucher aux ombres ni aux demi-tons.

Quand devrais-je utiliser le Boost?

Il y a deux raisons d’utiliser Boost. La première raison est d’obtenir les meilleurs résultats pour des images surexposées. Pour ces images, diminuez la valeur de Boost à 0,5 ou moins avant d’utiliser d’autres curseurs tels que Exposition ou Récupération. Cela vous permettra de minimiser la quantité d’exposition ou de récupération à appliquer, tout en maintenant l’image lumineuse. La deuxième raison d’utiliser Boost est d’obtenir une image plus plate (plus «brute») avant d’ajouter des réglages tels que Saturation.

Est-ce que RAW Power prend en charge la correction optique ?

Oui. Le moteur RAW d’Apple corrige automatiquement la distorsion et les ombres pour les appareils photo à objectifs fixes (Sony RX100, Leica Q, etc.). Il corrige également de nombreuses images dont les informations de profil d’objectif sont stockées dans les fichiers RAW (Fuji X-T20, par exemple). Cependant, il n’a pas de fonction de correction d’optique à usage général.

Supportez-vous le format DNG?

Oui. RAW Power prend en charge le format DNG, y compris les fichiers DNG linéaires et les fichiers RAW pour iPhone. Cependant, certains DNG de DJI ne sont pas pris en charge tels quels. Pour vérifier si vos DNG fonctionneront, ouvrez-les dans l’application de prévisualisation d’Apple pour Mac. S’ils fonctionnent dans Aperçu, RAW Power les prend également en charge. S’ils ne sont pas pris en charge, vous pouvez utiliser le convertisseur DNG d’Adobe pour les convertir par lots en une version de DNG prise en charge par RAW Power. Pour plus d’informations sur cette conversion par lots, veuillez nous contacter sur notre page de support.

Que sont les cercles à côté de l’histogramme?

Ces cercles s’illuminent chaque fois que des données de pixels sont surexposés dans ce canal (plus lumineuses que celles pouvant être affichées). Il existe des indicateurs pour le rouge, le vert et le bleu. Si vous cliquez sur l’un des cercles rouge, vert et bleu, la fonctionnalité «pixels surexposés» n’est activée que pour ce canal. Cliquez sur le cercle gris pour afficher les pixels chauds et froids, où rouge signifie un pixel surexposé en rouge, vert ou bleu, et bleu un pixel noir (0,0 dans les trois canaux).

Comment utiliser l’Extension RAW Power?

Une fois que RAW Power est téléchargé, vous pouvez activer l’extension dans les Préférences Système Extensions ou en cliquant sur le bouton Plus… dans la section Extensions du module Éditer de Photos d’Apple. Une fois que RAW Power est activé, vous pouvez le sélectionner dans la section Extensions du module Modifier de Photos d’Apple.

Où RAW Power conserve-t-il ses modifications?

RAW Power est un éditeur totalement non destructif. Dans l’application autonome, les modifications sont enregistrées dans un fichier annexe associé. Ce fichier associé se trouve dans le «dossier de sandbox» de l’application, plutôt que dans l’image originale, car RAW Power est une application en sandbox. Vous pouvez ensuite quitter RAW Power, rouvrir l’image ultérieurement et continuer là où vous l’avez laissée. Lorsque RAW Power s’exécute en tant qu’extension Photos, les modifications sont enregistrées dans la bibliothèque Photos. Vous pouvez également continuer là où vous en étiez, à condition de ne pas utiliser une autre extension (ou Photos elle-même) pour modifier l’image. Ceci est une limitation de l’application Photos.

Pourquoi ne vois-je pas de réglage Traitement RAW pour mon image?

Le traitement RAW apparaît uniquement pour les images RAW – pas pour les formats JPEG, TIFF ou autres formats d’image.

Lors de l’utilisation de l’extension Photos, il se peut que le réglage Traitement RAW ne soit pas affiché même si votre original est un fichier RAW. En effet, l’application Photos envoie un fichier JPEG à l’extension si l’image a déjà été modifiée dans Photos. Les modifications dans Photos incluent tout élément du module Modifier, y compris l’amélioration automatique, le recadrage, la rotation et les filtres. Pour éviter ce problème, appuyez sur le bouton Revenir à l’original dans Photos avant d’utiliser l’alimentation RAW.

Que fait la fonction Superposition dans Courbes?

Si la case « Superposition » est cochée, elle affiche les courbes rouge, verte et bleue ainsi que la courbe combinée (lorsque vous regardez la courbe combinée). Cela ne fait rien si vous ne regardez que les courbes rouge, verte ou bleue. Cette fonctionnalité vous permet de voir en un coup d’œil comment les quatre courbes sont appliquées.
