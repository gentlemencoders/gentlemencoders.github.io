RAW Power for macOS Release Notes

Version 3.4.13

  • (macOS 11+) Añadida compatibilidad con OM Digital OM-5, Sony FX30 y Samsung NX500. Ver lista completa
  • (macOS 13+) Añadida compatibilidad con Canon R6 II. Ver lista completa
  • Corrección de los principales fallos.
  • Mejoras de rendimiento y correcciones generales.

Version 3.4.12

  • (macOS 11+) Añadir soporte para Sony A7 R5. Ver lista completa
  • Solución del error de Ventura que provoca un bloqueo al utilizar el cuentagotas en la Extensión Fotos.
  • Solución de otro error de Ventura con Auto Enhance que causa una imagen corrupta temporalmente.
  • Añadir texto de ayuda localizado para imágenes no soportadas.

Version 3.4.11

  • (macOS 13) Añade la Fujifilm X-H2 y la Fujifilm X-T5 a la lista de cámaras compatibles. Ver lista completa
  • (macOS 11+) Añade la Olympus SH-3 a la lista de cámaras compatibles. Ver lista completa
  • Corregir error al detectar el modelo de lente para la corrección automática de la misma.
  • Corrección de un error en la exportación y en las imágenes enderezadas.
  • Otras correcciones y mejoras.

Version 3.4.10

  • (macOS 11+) Añadir Fujifilm X-H2S y Canon R7 a la lista de cámaras compatibles. Ver lista completa
  • Escribir los metadatos completos al exportar TIFFs desde la Extensión de Fotos.
  • Otras correcciones y mejoras.

Version 3.4.9

  • (macOS 11+) Añade Z9 a la lista de cámaras compatibles. Ver lista completa
  • Mejore la compatibilidad con LUT de múltiples programas de edición.

Version 3.4.8

  • (macOS 11+) Se añaden Olympus OM-1 y Nikon Z fc a la lista de cámaras compatibles. Ver lista completa
  • Mejore la compatibilidad con LUT de múltiples programas de edición.

Version 3.4.5

  • (macOS 11+) Se añaden las cámaras Fujifilm GFX 50S II, Sony A7R 3A, A7R 4A, Sony ZV-E10, Pentax K3 Mark III (sólo DNG) y Ricoh GR IIIx a la lista de cámaras compatibles. Ver lista completa
  • Nuevo: Modo de Vista Previa Rápida para moverse velozmente a través de las imágenes cuando se usa el Visor (en Preferencias).
  • (macOS 11+) Compatible con RAW comprimido sin pérdidas Sony (sólo imágenes RAW extendido).

Version 3.4.3

  • (macOS 11+) Se añaden las cámaras Sony A7 IV, Samsung NX300 y Fujifilm X-T30 II a la lista de cámaras compatibles. Ver lista completa
  • Se añade la opción de 150 ppp al Exportar.
  • Se ha corregido un error que provocaba que la aplicación ocasionalmente abriese una pestaña extra cada vez que se iniciaba.
  • Se ha traduccido la aplicación al Español.
